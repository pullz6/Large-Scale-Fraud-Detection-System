# Fraud_Detection

Financial institutions and e-commerce platforms face a growing threat from **fraudulent transactions**, which result in **significant revenue loss** (estimated at $42 billion globally in 2023). Traditional rule-based fraud detection systems:

- **Fail to scale** with increasing transaction volumes (1TB+ daily).
- **Struggle with imbalanced data** (fraud cases are rare, e.g., < 0.1% of transactions).
- **Lack real-time responsiveness**, allowing fraudsters to exploit delays.

This project aims to build an end-to-end MLOps pipeline to detect fraud at scale for the above.
